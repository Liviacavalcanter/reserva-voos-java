# Sistema de Reserva de Voos (Console)

Projeto desenvolvido em Java com foco na aplicação prática dos conceitos de Programação Orientada a Objetos (POO).  
Este sistema simula um ambiente de reservas de voos, com funcionalidades completas de gerenciamento de dados no terminal.

## Descrição
O sistema permite realizar as operações básicas de um CRUD (Criar, Ler, Atualizar e Deletar) para cinco entidades principais: Clientes, Voos, Reservas, Pagamento e Assento.

A interação ocorre por meio de um menu numérico no console, permitindo a navegação entre as opções de forma simples e direta.

## Conceitos Aplicados
- Princípios de POO (Classes, Objetos, Encapsulamento, Herança e Polimorfismo)
- Estruturação de dados em memória
- Modularização e boas práticas de codificação

## Tecnologias Utilizadas
- Java 17
- Execução via terminal/console
- IDE: IntelliJ
